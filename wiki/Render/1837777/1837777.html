Library that provides .net developers tools,
* To query,control game servers remotely.
* Find servers that match certain criteria.
* Query Steam web api easily.

Features
  • Query any Source/Gold Source servers to get basic information of server,information of players currently playing in server and server rules.(Server Query Protocol)
  • Send commands to server using Rcon password.(Rcon Protocol)
  • Listen to logs and/subscribe to different events that are logged.(Logged Events)
  • Query Master Server(also supports filtering).(Master Server Query Protocol)
  • Query Steam web api (Web api)
  • Has utilities like Steam id converter.
